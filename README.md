🎓 Student Management Telegram Bot
Цей проєкт є багатофункціональним Telegram-ботом для студентів та адміністраторів, розробленим на базі бібліотеки aiogram 3.x. Бот автоматизує відстеження дедлайнів, перевірку оцінок, керування розкладом та синхронізацію даних із зовнішніми Google Таблицями.

✨ Основні можливості
📊 Робота з оцінками та успішністю
Отримання оцінок: Користувачі можуть переглядати свій середній бал по всіх предметах або отримувати детальний список оцінок за кожну дисципліну.
Рейтинги: Функція розрахунку рейтингу групи за конкретним предметом на основі середнього балу.
Синхронізація з Google Sheets: Модуль автоматичного парсингу даних із Google Таблиць для оновлення списків студентів, їхніх оцінок та актуальних завдань.
⏳ Дедлайни та завдання
Відстеження дедлайнів: Перегляд усіх актуальних завдань, збережених у системі.
«Гарячі» дедлайни: Спеціальна функція сповіщення про завдання, термін здачі яких добігає кінця (менше або дорівнює 2 дням).
📅 Розклад та сповіщення
Розумний розклад: Підтримка логіки «Чисельник/Знаменник» на основі ISO-номера тижня.
Щоденний огляд: Автоматичне надсилання розкладу на день о 08:30.
Нагадування про пари: Бот надсилає нагадування за 10 хвилин до початку кожного заняття.
🛠 Адміністрування та групи
Керування групами: Реєстрація користувачів із закріпленням за конкретною академічною групою (наприклад, ПЗ-23-1/9).
Адмін-розсилка (Broadcast): Можливість надсилати важливі повідомлення як усім зареєстрованим користувачам, так і окремим групам.
Збір Chat ID: Автоматичне збереження ідентифікаторів користувачів для розсилок.

🛠 Технологічний стек
Python 3.10+


aiogram 3.x — асинхронний фреймворк для Telegram-ботів.
Pandas & Openpyxl — для парсингу даних із Google Таблиць.
APScheduler — для планування щоденних розкладів та нагадувань.
JSON — використання локальних файлів як бази даних (data.json, groups.json, schedule.json).

📂 Структура проєкту
main.py — точка входу, запуск бота.
commands.py — обробка основних текстових команд та логіки взаємодії.
db_manager.py — логіка роботи з дедлайнами, рейтингами та оцінками.
sync_sheets.py — модуль для синхронізації з Google Sheets.
scheduler_logic.py — автоматизація розкладу та нагадувань.
admin_broadcast.py — функціонал адміністративної розсилки.
group_manager.py / group_selector.py — логіка реєстрації та вибору груп.
std_keyboard.py — конфігурація клавіатур бота.

🚀 Налаштування та запуск
Встановлення залежностей:
Bash
pip install aiogram pandas requests openpyxl apscheduler pytz




Конфігурація:
У файлі settings.py вкажіть ваш BOT_TOKEN.
У файлах admin_broadcast.py та group_manager.py оновіть ADMIN_ID на ваш власний Telegram ID.
Для синхронізації переконайтеся, що sheet_id у sync_sheets.py відповідає вашій таблиці.
Запуск:
Bash
python main.py
